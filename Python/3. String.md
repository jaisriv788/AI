## Introduction

- In **Python**, strings are a sequence of Unicode Characters.
- Below are the ways to declare a string :

```python
s1 = 'Hello'
s2 = "Hello"
s3 = '''Hello'''
s4 = """Hello"""

print(s1,s2,s3, sep="\n")
```

## Accessing Substrings From A String

```python (exec)
s = "hello"

# Positive Indexing
print(s[0])
print(s[1])
print(s[2])
print(s[3])
print(s[4]) 

# Negative Indexing
print(s[-1])
print(s[-2])
print(s[-3])
print(s[-4])
print(s[-5])
```

### Slicing

```python (exec)
s = "hello world"

print(s[0:7])
```
