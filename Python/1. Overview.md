## Data Types
1. **Integers** : Python can handle large integers within it self and i.e. up to 1e309(10^308).
2. **Decimals**: Limit for the it is 1.7e309(1.7^309).
3. **Boolean**: It either will be **True** or **False**.
4. **String**: Its form of text.
5. **Complex Number**: Its also handled by python but are used less.
6. **List**: Its just like array and is represented as [1,2,3,4].
7. **Tuple**: Its also like array but are represented with round brackets like (1,2,3,4).
8. **Sets**: Its like set in math and and is represented like {1,2,3,4}.
9. **Dictionary**: Its always written in the form of key value pair just like objects.                            e.g. {"name": "Nitish", "gender": "Male"}.
